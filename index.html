A continuación se presentan proyectos y propuestas vinculados a gestión de datos, documentación digital y análisis operacional, alineados con mi formación en Programación y Análisis de Sistemas.

Sistema de Gestión de Tareas (SGTareas)

Sistema web desarrollado en C# para la gestión eficiente de tareas en equipos de trabajo. Implementa arquitectura de 3 capas, gestión de usuarios, roles, asignación de tareas y seguimiento de estados (Pendiente, En Progreso, Finalizada).

Tecnologías: C# .NET, SQL Server, HTML5, CSS3, JavaScript.

Ver repositorio en GitHub

Panel de Control para Inventarios Logísticos

Diseño de un panel web para visualizar inventarios, movimientos de bodega y trazabilidad de productos, basado en la experiencia en logística y operaciones. El objetivo es apoyar la toma de decisiones mediante indicadores simples.

Tecnologías: HTML5, CSS3, JavaScript (DOM básico), hojas de cálculo.
