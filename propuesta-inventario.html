Panel de Control Logístico

Propuesta de Desarrollo Tecnológico

Descripción del Proyecto

Este proyecto consiste en el diseño y desarrollo de un Panel de Control (Dashboard) Web orientado a la gestión de inventarios y logística. La propuesta nace de la necesidad de visualizar de manera clara y rápida el estado de los stocks, movimientos de entrada/salida y alertas de abastecimiento en entornos operativos.

Objetivos

  • Centralizar la información de inventario en una interfaz visual amigable.
  • Facilitar la toma de decisiones mediante indicadores clave (KPIs) simples.
  • Reducir errores en el registro de movimientos mediante formularios validados.
  • Proveer trazabilidad básica de productos (quién retiró, cuándo y para qué).

Funcionalidades Propuestas (MVP)

Para la primera versión (Producto Mínimo Viable), se contemplan las siguientes características:

  • Dashboard Principal: Gráficos de resumen (Stock total, Artículos críticos).
  • Tabla de Inventario: Listado filtrable y ordenable de productos.
  • Registro de Movimientos: Formularios para "Ingreso" y "Salida" de mercadería.
  • Alertas Visuales: Indicadores de color cuando un producto está bajo el stock mínimo.

Tecnologías

HTML5 Semántico CSS3 (Grid/Flexbox) JavaScript Vanilla Chart.js (Librería de gráficos) JSON / CSV (Almacenamiento local simulado)
